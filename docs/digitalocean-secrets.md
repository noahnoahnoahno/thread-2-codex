# DigitalOcean 환경 변수

이 프로젝트는 YouTube/Drive OAuth 토큰을 GitHub에 올리지 않습니다. DigitalOcean App Platform의 앱 환경 변수에 아래 값을 `Secret` 타입으로 넣어야 합니다.

로컬에서 값 목록을 만들 때:

```bash
python3 scripts/print_digitalocean_env.py
```

DigitalOcean에 넣을 값:

- `UPLOADER_ADMIN_TOKEN`: 웹 대시보드 버튼 실행용 관리자 토큰
- `UPLOADER_SECRET_AUTO_UP_CREDENTIALS_JSON_B64`
- `UPLOADER_SECRET_AUTO_UP_TOKEN_JSON_B64`
- `UPLOADER_SECRET_DRIVE_TOKEN_JSON_B64`
- `UPLOADER_SECRET_NINGNING_YOUTUBE_CREDENTIALS_JSON_B64`
- `UPLOADER_SECRET_MOSONGEEAI_YOUTUBE_CREDENTIALS_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_DAPJEONGSA_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_NANGMAN_TONGSINSA_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_NINGNING_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_AMUSEASIA_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_VOGUE_CITY_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_TWOSOME_MOVIE_JSON_B64`
- `UPLOADER_SECRET_YOUTUBE_MOSONGEEAI_JSON_B64`

앱은 시작 후 환경 변수 값을 `/app/secrets/*.json` 파일로 복원해서 기존 코드가 그대로 읽도록 동작합니다.

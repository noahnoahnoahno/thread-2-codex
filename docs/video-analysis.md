# YouTube 영상 자막 기반 분석

대상 영상: https://www.youtube.com/watch?v=sORoHYP7HRU

자막 파일: `youtube_sORoHYP7HRU_transcript_ko.txt`

## 핵심 요약

영상에서 소개하는 프로그램은 `아크 클리퍼` 형태의 롱폼-투-쇼츠 자동 제작 도구다. 사용자가 유튜브 URL을 입력하면 시스템이 영상을 다운로드하고, 오디오를 추출하고, 음성 인식과 AI 분석을 거쳐 후킹 구간을 최대 6개 정도 추천한다. 사용자는 각 클립의 시작/끝 지점, 제목, 자막, 해시태그, 레이아웃, 폰트, 색상, 제목/채널명 위치를 조정한 뒤 MP4로 생성 및 다운로드한다.

## 자막에서 확인된 주요 기능

1. 유튜브 롱폼 URL 입력
2. 대상 영상 확인 모달
3. 5단계 처리 진행 표시
   - 영상 다운로드
   - 오디오 추출
   - 음성 인식
   - AI 분석
   - 종합 생성
4. AI 기반 후킹 구간 탐지
5. 최대 6개 클립 후보 생성
6. 각 클립별 길이 표시
7. 후킹 구간 선정 근거 설명
8. 자동 제목 생성
9. 추천 유튜브 해시태그 생성
10. 해시태그 수동 추가
11. 쇼츠 미리보기
12. 세이프존/3분할 가이드 표시
13. 레터박스 레이아웃
14. 세로 크롭 레이아웃
15. 제목 위치 조정
16. 채널명 표시 및 위치 조정
17. 폰트 변경
18. 색상 변경
19. 클립 시작/끝 지점 조정
20. -1초/+1초 단위 미세 조정
21. 선택 클립 단일 생성
22. 복수 클립 일괄 생성
23. MP4 다운로드

## 가능 여부

구현은 가능하다. 다만 완성도를 좌우하는 핵심 난이도는 UI보다 영상 처리 파이프라인과 AI 구간 선택 품질이다.

### 현실적으로 바로 가능한 범위

- 사용자가 직접 업로드한 영상 또는 사용 권한이 있는 영상 처리
- URL 기반 메타데이터 확인
- 오디오 추출
- Whisper 기반 전사
- LLM 기반 후킹 구간 후보 추천
- 후보별 제목/해시태그/선정 근거 생성
- 세로 쇼츠 MP4 렌더링
- 레터박스/세로 크롭 템플릿
- 간단한 자막 입히기
- 클립 시작/끝 조정
- 생성 파일 다운로드

### 주의가 필요한 범위

- 임의 유튜브 영상 다운로드는 저작권과 YouTube 약관 이슈가 있을 수 있다. MVP는 `사용자 소유/사용 허가 영상` 또는 `파일 업로드`를 기본 경로로 두는 편이 안전하다.
- 화자 중심 자동 크롭은 난이도가 높다. 초기에는 중앙 크롭/수동 위치 조정으로 시작하고, 이후 얼굴/화자 추적을 추가한다.
- 웹에서 긴 영상 여러 개를 동시에 처리하면 서버 비용과 처리 시간이 커진다. 작업 큐와 스토리지 만료 정책이 필요하다.
- AI가 뽑은 후킹 구간은 항상 검수 UI가 필요하다. 완전 자동 업로드는 초기에 권장하지 않는다.

## 추천 MVP

MVP 목표는 "사용 권한이 있는 롱폼 영상 하나를 업로드하거나 URL로 가져와 3~6개의 쇼츠 후보를 만들고, 사용자가 고른 후보를 MP4로 렌더링하는 것"이다.

필수 기능:

- 파일 업로드 또는 허용된 URL 입력
- 영상/오디오 분석 작업 생성
- Whisper 전사
- LLM 후보 구간 추천
- 후보 카드 목록
- 후보별 타임라인 조정
- 제목/해시태그 편집
- 레터박스 또는 세로 크롭 선택
- 자막 스타일 최소 설정
- MP4 렌더링 및 다운로드

후순위 기능:

- 얼굴/화자 추적 자동 크롭
- 브랜드 템플릿 저장
- 다중 영상 배치 처리
- YouTube Shorts 자동 업로드
- 팀/로그인/과금
- 댓글/성과 분석 기반 재학습

## 핵심 성공 기준

- 10분 내외 영상 기준, 3~6개 후보를 안정적으로 생성한다.
- 후보의 50% 이상이 사람이 보기에도 쇼츠로 쓸 만하다.
- 1개 클립 렌더링 시간이 1분 이하로 유지된다.
- 생성 MP4가 9:16, 1080x1920, H.264/AAC로 정상 출력된다.
- 제목/자막이 쇼츠 UI 세이프존을 침범하지 않는다.
